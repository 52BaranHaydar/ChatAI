# ChatAI 💬

A real-time AI-powered chat app built with SwiftUI, Firebase, and Sign in with Apple.

## Features
- 🍎 Sign in with Apple — secure authentication
- 💬 Real-time messaging with Firebase Firestore
- 🤖 AI sentiment analysis — detects message emotion (😊😔😐)
- 🔒 Secure & private with Firebase Auth
- ⚡ Live updates with Firestore listeners

## Tech Stack
- **SwiftUI** — Modern declarative UI
- **Firebase Auth** — Sign in with Apple
- **Firebase Firestore** — Real-time database
- **MVVM Architecture** — Clean code structure
- **Swift Concurrency** — async/await

## Requirements
- iOS 17+
- Xcode 15+
- Swift 5.9+
- Physical device for Sign in with Apple

## Installation
1. Clone the repo
```bash
   git clone https://github.com/52BaranHaydar/ChatAI.git
```
2. Open `ChatAI.xcodeproj` in Xcode
3. Add your `GoogleService-Info.plist`
4. Build and run on a physical device
